# BookLens2 웹사이트 소개 자료

## 📚 웹사이트 개요

### 웹사이트 목적
**BookLens2**는 독서를 즐기는 모든 사람을 위한 종합 독서 관리 플랫폼입니다. 
- 개인의 독서 여정을 체계적으로 관리하고 추적
- 독서 후기를 공유하며 커뮤니티와 소통
- 책 정보 검색 및 도서관/서점 위치 확인
- 독서 습관 형성 및 독서 통계 관리

### 핵심 가치
- **개인화된 독서 관리**: 나만의 독서 라이브러리 구축
- **커뮤니티 기반 학습**: 다른 독서자들과의 지식 공유
- **편리한 정보 접근**: 책 검색부터 구매/대출 위치까지 원스톱 서비스
- **데이터 기반 인사이트**: 독서 시간, 완독률 등 통계 제공

## 🎯 주요 기능

### 1. 독서 관리 시스템
- 책 추가 및 독서 진행 상황 추적
- 실시간 독서 시간 측정
- 독서 진행률 시각화
- 완독 책 관리

### 2. 커뮤니티 기능
- 독서 후기 작성 및 공유
- 평점 및 태그 시스템
- 포스팅 검색 및 필터링
- 다른 사용자들의 독서 경험 탐색

### 3. 책 정보 검색
- Google Books API 연동
- ISBN, 제목, 저자로 책 검색
- 상세 책 정보 제공

### 4. 위치 기반 서비스
- 카카오맵 연동
- 도서관/서점 위치 표시
- 책별 재고 정보 확인
- 대출 가능/구매 가능 여부 표시

### 5. 시각적 컬렉션
- 책 표지 갤러리
- 장르별, 시리즈별 필터링
- 독서 진행률 요약

## 📄 페이지별 상세 기능

### 1. 홈페이지 (HomePage)
**기능:**
- **히어로 배너**: 웹사이트 소개 및 주요 프로모션 슬라이드
- **베스트셀러 섹션**: 현재 인기 있는 책 6권 표시
- **신간 도서 섹션**: 최신 출간 도서 4권 추천
- **프로모션 배너**: 회원 가입 유도 및 서비스 소개

**특징:**
- 반응형 그리드 레이아웃
- 책 표지 이미지 및 기본 정보 표시
- 각 책 클릭 시 상세 페이지로 이동

---

### 2. 마이라이브러리 (MyLibraryPage)
**핵심 기능:**
- **책 추가**
  - ISBN 검색: ISBN 입력으로 책 정보 자동 불러오기
  - 제목/저자 검색: Google Books API를 통한 실시간 검색
  - 수동 입력: 직접 책 정보 입력
  - 책 표지 미리보기

- **독서 진행 관리**
  - 독서 시작/종료 기능
  - 실시간 독서 시간 측정 (타이머)
  - 읽은 페이지 수 입력
  - 진행률 자동 계산 및 시각화
  - 독서 상태 관리 (읽는 중/완독)

- **통계 대시보드**
  - 전체 도서 수
  - 읽는 중인 책 수
  - 완독한 책 수
  - 총 독서 시간
  - 오늘 읽은 시간
  - 내 포스팅 개수

- **완독 책 관리**
  - 완독한 책 목록 표시
  - 완독일 기록
  - 포스팅 작성 바로가기

- **책 정보 관리**
  - 책 삭제
  - 메모 작성
  - 시작일/완독일 기록

**UI/UX 특징:**
- 카드 기반 레이아웃
- 진행률 프로그레스 바
- 실시간 시간 표시
- 모달을 통한 독서 세션 관리

---

### 3. 커뮤니티 (CommunityPage)
**핵심 기능:**
- **포스팅 목록**
  - 모든 사용자의 독서 후기 표시
  - 그리드 레이아웃으로 카드 형태 표시
  - 책 정보, 평점, 태그, 작성자 정보 포함

- **검색 및 필터링**
  - 책 제목, 저자, 내용, 태그로 검색
  - 정렬 옵션: 최신순, 평점순, 오래된순
  - 내 포스트만 보기 필터

- **포스팅 상세**
  - 제목, 내용 미리보기
  - 평점 (별점 1-5점)
  - 태그 표시
  - 작성 시간 표시 (상대 시간: "3시간 전" 등)
  - 완독일 정보

- **포스팅 관리**
  - 내 포스팅 수정
  - 내 포스팅 삭제
  - 포스팅 상세 보기

**특징:**
- 반응형 3열 그리드
- 실시간 검색 (디바운싱 적용)
- localStorage 기반 데이터 저장

---

### 4. 갤러리 (GalleryPage)
**핵심 기능:**
- **책 표지 컬렉션**
  - Google Books API에서 출판사별 책 가져오기
  - Masonry 레이아웃으로 책 표지 표시
  - 무한 스크롤 애니메이션 (물레방아 효과)

- **필터링**
  - 장르별 필터
  - 시리즈별 필터
  - 연도별 필터

- **진행률 요약**
  - 읽은 책 / 전체 책 비율 표시
  - 시각적 프로그레스 바

**특징:**
- 시각적으로 아름다운 책 표지 갤러리
- 부드러운 스크롤 애니메이션
- 반응형 레이아웃

---

### 5. 지도 (MapPage)
**핵심 기능:**
- **책 검색**
  - Google Books API를 통한 책 검색
  - 검색 결과에서 책 선택

- **위치 표시**
  - 카카오맵 연동
  - 도서관/서점 위치 마커 표시
  - 선택한 책의 재고가 있는 위치만 필터링

- **재고 정보**
  - 각 위치별 책 재고 수량 표시
  - 대출 가능/구매 가능 권수
  - 서점의 경우 가격 정보 표시
  - 도서관의 경우 대출 가능 여부

- **필터링**
  - 전체 / 도서관만 / 서점만 필터
  - 목록 보기/숨기기 토글

- **인포윈도우**
  - 마커 클릭 시 상세 정보 팝업
  - 위치명, 주소, 재고 정보 표시
  - 닫기 버튼

**특징:**
- 실시간 지도 업데이트
- 인터랙티브 마커
- 위치별 상세 정보 제공

---

### 6. 포스팅 작성 (PostingPage)
**핵심 기능:**
- **포스팅 작성**
  - 제목 입력 (선택사항, 기본값: "책제목 독후감")
  - 평점 선택 (별점 1-5점)
  - 내용 작성 (필수)
  - 태그 입력 (쉼표로 구분)

- **포스팅 수정**
  - 기존 포스팅 수정 모드
  - 기존 내용 불러오기
  - 수정 후 커뮤니티로 이동

- **책 정보 표시**
  - 완독한 책 정보 카드
  - 책 제목, 저자, 완독일 표시

**특징:**
- 직관적인 폼 UI
- 실시간 글자 수 표시
- 완독한 책만 포스팅 가능

---

### 7. 책 상세 페이지 (BookDetailPage)
**핵심 기능:**
- **책 정보 표시**
  - 책 표지 이미지
  - 제목, 부제목
  - 저자, 출판사, 발행일
  - ISBN, 페이지 수
  - 평균 평점 및 평가 수

- **탭 기반 상세 정보**
  - 책 소개: 설명 및 카테고리
  - 목차: 책 목차 정보
  - 저자 소개: 저자 정보 및 Google Books 링크
  - 추천사: 미리보기 및 추가 정보 링크

- **외부 링크**
  - 교보문고, 알라딘, YES24, 영풍문고 바로가기
  - Google Books 링크

**특징:**
- Google Books API 데이터 활용
- 탭 네비게이션으로 정보 구조화
- 반응형 레이아웃

---

### 8. 로그인/회원가입 (LoginPage / SignupPage)
**핵심 기능:**
- **로그인**
  - 이메일/비밀번호 로그인
  - 로그인 상태 유지 옵션
  - 소셜 로그인 (Google, Naver) - UI만 구현
  - 비밀번호 찾기 링크

- **회원가입**
  - 이메일/비밀번호 회원가입
  - 비밀번호 확인
  - 이용약관 동의
  - 소셜 회원가입 (Google, Naver) - UI만 구현

**특징:**
- 깔끔한 카드 디자인
- 폼 유효성 검사
- 에러 메시지 표시

## 🛠 기술 스택

### Frontend
- **React**: 사용자 인터페이스 구축
- **Vite**: 빠른 개발 환경 및 빌드 도구
- **React Router**: 페이지 라우팅
- **Tailwind CSS**: 스타일링
- **Context API**: 전역 상태 관리 (인증)

### External APIs
- **Google Books API**: 책 정보 검색 및 상세 정보
- **Kakao Map API**: 지도 및 위치 서비스

### 데이터 저장
- **localStorage**: 클라이언트 사이드 데이터 저장
  - 사용자 인증 정보
  - 마이라이브러리 책 목록
  - 독서 세션 정보
  - 커뮤니티 포스팅
